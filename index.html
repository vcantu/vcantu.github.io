NovaKey - Android App

This is my pride and joy. I have been working on this app since September of 2014. I always found the idea of keyboards on touchscreens backwards. Since I had some Java experience I decided to jump in and try it for myself.

The keyboard consists of six areas, five outer ones and one inner. Each area is composed of one main, tappable, key and various other, swipeable, keys. Because every key is a unique gesture there it is much harder to make mistakes.

Because it's compactability NovaKey includes many useful features that most generic QWERTY keyboards don't have. These include:


  • Place Picker
    Android Rolodex View
    Small Java Calculator
    Java Number to English
